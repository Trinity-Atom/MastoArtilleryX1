# Summary

### What is it?
This is a small recycling bin that looks like the Windows 10 recycling bin on the desktop.
The dimensions are 116x126x150mm. You can print it as a multicolor or singlecolor print.

### How to print?
If printed as a singlecolor print you have to print the bin and the arrows separately and glue them afterwards. I designed an alignment tool for easy alignment before glueing it together.
The arrows for the multicolor print are smaller so that you can print it without support.

### Remarks
If you have problems with printing or other remarks to the design, please let me know.
